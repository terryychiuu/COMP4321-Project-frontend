# COMP4321-Project-frontend

The project is the course project for the HKUST COMP 4321 course. It is a web search engine based on vector space model.

## Building & running

### `npm install`

To install all the required dependency

### `npm start`

Runs the app in the development mode.\
Visit the development server at http://localhost:3000 with your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.
